To navigate the town of Steelport type [N]orth, [W]est, [E]ast, [S]outh, [Forward], [Left], [Right], [Backward].
If you are in one of the many buildings of Steelport type [Up], [Down] or [Ascend], [Descend] to change floors.
While in a location; type [Get] "item name" to pickup different items.
To drop an item type [Drop] "item name".
If you want to view the inventory, type [Inventory].
